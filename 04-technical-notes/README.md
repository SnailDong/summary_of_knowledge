# Technical Notes

这个目录收录技术学习笔记，按主题拆分，方便后续查找和面试复习。

## 分类

- `java/`：Java 基础、JVM、集合、异常、反射、泛型、注解。
- `android-ui/`：View 绘制、事件分发、自定义 View、RecyclerView、Jetpack、WebView 等。
- `android-framework/`：Android Framework、Handler、Binder、ANR 等系统机制。
- `performance/`：Android 性能优化、布局优化、内存和卡顿分析。
- `open-source-libraries/`：Glide、OkHttp、Retrofit、RxJava 等开源库。
- `database-network/`：网络、数据库和 TCP/IP 相关笔记。

## 整理约定

- 正文使用标准 Markdown，不保留导出工具生成的字体颜色、样式标签和 OCR 注释。
- 代码片段保留 fenced code block 或行内代码格式。
- 图片链接保留在原文位置，方便后续需要时继续查看图示。
