Architecture Overview — Knowledge Site

目标

使用 Astro + Markdown 将当前个人知识库导出为可部署、可手机访问的静态 HTML 网站。

组件职责

组件职责
Astro pages首页、个人介绍、项目经验、技术笔记、AI Workflow 文章路由。
sync-content.mjs从父目录知识库同步公开 Markdown 和简历资源。
src/content构建时 Markdown 内容副本。
public/resumePDF 与 HTML 简历等静态资源。
dist构建后可部署的纯静态网站。

Mermaid

flowchart LR
  A[Source Markdown] --> B[scripts/sync-content.mjs]
  B --> C[src/content]
  B --> D[public/resume]
  C --> E[Astro Routes]
  D --> E
  E --> F[dist static HTML]
  F --> G[Phone / Desktop Browser]

公开边界

默认只接入 `01-resume/current`、`03-project-experience` 顶层项目总结、`04-technical-notes`、`05-ai-workflow-skills`。`02-interview`、`99-archive`、项目内 `skills/` 证据材料默认不发布。
