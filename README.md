# Summary of Knowledge

这个目录用于集中管理个人简历、面试材料、项目经验、技术笔记和 AI workflow skills。

## 目录结构

```text
summary_of_knowledge/
├── 01-resume/
├── 02-interview/
├── 03-project-experience/
├── 04-technical-notes/
├── 05-ai-workflow-skills/
├── 06-packages-and-exports/
└── 99-archive/
```

## 01-resume

简历和投递材料。

- `current/`：当前主用简历源文件、HTML 排版和 PDF。
- `generated/`：结构化或自动生成的导出版本。
- `drafts/`：草稿、临时版本或 Word 文档。

## 02-interview

面试准备材料。

- `current/interview-reference.md`：当前主用面试问答手册。
- `current/abilities.md`：个人能力提炼和表达材料。
- `question-bank/`：后续可放专项题库或追问整理。

## 03-project-experience

项目经历和工作总结。

这里收录 Warranty Plus、Provisioning Kit、设备绑定、Homebase 新品绑定等项目总结，以及项目中使用过的 workflow/skill 证据材料。

## 04-technical-notes

技术学习笔记。

- `java/`：Java 基础、JVM、集合、异常、反射、泛型、注解。
- `android-ui/`：View、事件、RecyclerView、Jetpack、Material Design、WebView 等。
- `android-framework/`：Android Framework 相关笔记。
- `performance/`：性能优化专题。
- `open-source-libraries/`：Glide、OkHttp、Retrofit、RxJava 等开源框架。
- `database-network/`：网络和数据库相关笔记。

## 05-ai-workflow-skills

AI workflow 和 skills 材料。

- `custom-skills/`：自定义工作流 skills，例如 architect、code-review、dev-workflow、story-craftsman。
- `superpowers/`：superpowers 系列 skills。

## 06-packages-and-exports

压缩包、导出包和迁移包。

## 99-archive

历史版本和归档材料。

- `resume-backups/`：简历历史备份。
- `interview-backups/`：面试材料历史备份。
- `old-resume/`：旧版简历。
- `old-interview-references/`：旧版面试参考材料。
- `misc/`：暂时无法归入主目录的杂项或旧说明。
